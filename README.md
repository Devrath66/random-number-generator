Random Generator Website
A simple, interactive web application that lets users generate:
- A 6-digit OTP
- A random dice roll (1–6)
- A random number (1–100000)
Built using HTML, CSS, and JavaScript, this project transforms a C++ console program into a visually appealing and responsive website.

Features
- One-click generation of OTPs, dice rolls, and random numbers
- Responsive design for desktop and mobile
- Modern UI with gradient backgrounds, icons, and hover effects
- No dependencies — runs entirely in the browser

1. Clone or Download
git clone https://github.com/your-username/random-generator-website.git

2. Open in Browser
Double-click micro.html or open it with any modern browser (Chrome, Edge, Firefox).

Technologies Used
HTML : structure and layout
CSS : Styling and responsiveness
JavaScript : Logic and interactivity
Google Fonts : Custom typography (Roboto)
